
# Impulse API

A e-commerce API for basic e-commerce needs.


## Run Locally

Clone the project

```bash
  git clone https://github.com/mustafashen/impulse-api/tree/main
```

Go to the project directory

```bash
  cd impulse-api
```

Install dependencies

```bash
  npm install
```
Build TypeScript

```bash
  npm run build
```
Start the server

```bash
  npm run start
```

Start the server in dev

```bash
  npm run dev
```

Testing

```bash
  npm run test
```

Testing concurrently

```bash
  npm run test:watch
```
